# Emotion Detection - AI Web Application

A web application that detects emotions from text input using AI, 
built with Python and Flask.

## Tech Stack

- Backend: Python + Flask
- AI/NLP: Emotion Detection API
- Frontend: HTML + CSS + JavaScript

## Features

- Input text and detect emotions
- Returns scores for: anger, disgust, fear, joy, sadness, surprise
- Displays dominant emotion
- Error handling for blank inputs
- Unit tested functions

## Run Locally

1. Clone the repo
2. Run: pip install -r requirements.txt
3. Run: python server.py
